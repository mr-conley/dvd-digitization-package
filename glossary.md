Term	Definition

Transcoding	The process of converting a video file from one format (e.g., MPEG-2 from a DVD) to a more modern, compressed format (e.g., H.264/MP4).

Container	A file format (like .mp4 or .mkv) that holds video data, audio data, and metadata together in one package.

Deinterlacing	A digital process used to convert interlaced video (the "striped" look often found on older DVDs) into a smooth, progressive format for modern displays.

exFAT	A file system optimized for flash drives that allows for single files larger than 4GB, which is often necessary for high-quality video exports.

Handbrake	The open-source transcoding software used in this SOP to convert physical media into digital files.

Bitrate	The amount of data processed per second in a video file; a higher bitrate usually means better quality but a larger file size.
